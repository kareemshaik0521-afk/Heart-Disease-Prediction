# ❤️ Heart Disease Prediction System

A Machine Learning based web application that predicts the likelihood of heart disease using patient health parameters.  
The system uses a trained ML model and provides an easy-to-use interface where users can input medical details and receive predictions.

---

## 📌 Project Overview
Heart disease is one of the leading causes of death worldwide. Early detection can help in prevention and treatment.  
This project uses machine learning techniques to analyze medical data and predict whether a person is at risk of heart disease.

---

## 🚀 Features
- Predicts heart disease risk based on medical inputs
- Data preprocessing and feature selection
- Machine Learning model trained using **Scikit-learn**
- Interactive **Streamlit web interface**
- Fast and simple user experience
- Displays prediction results instantly

---

## 🛠️ Technologies Used
- **Python**
- **Scikit-learn**
- **Pandas**
- **NumPy**
- **Streamlit**
- **Machine Learning (Random Forest Classifier)**

---

## 📊 Machine Learning Model
The prediction system uses the **Random Forest Classifier**, which improves accuracy by combining multiple decision trees and reducing overfitting.

Steps involved:
1. Data preprocessing
2. Feature selection
3. Model training
4. Model evaluation
5. Prediction through user input

---

## 🖥️ User Interface
The application is built using **Streamlit**, allowing users to:
- Enter medical parameters such as age, cholesterol, blood pressure, etc.
- Click a button to generate prediction
- Instantly view whether the patient is likely to have heart disease

## ▶️ How to Run the Project

1. Clone the repository
```
git clone https://github.com/yourusername/heart-disease-prediction.git
```

2. Navigate to the project folder
```
cd heart-disease-prediction
```

3. Install required libraries
```
pip install -r requirements.txt
```

4. Run the Streamlit application
```
streamlit run app.py
```
